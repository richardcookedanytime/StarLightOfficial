## 使用方法
```bash
# 安装
curl -fsSL https://starlight.io/install.sh | bash

# 创建项目
starlight create my-project && cd my-project

# 编译运行
jstarc src/main.sl --target=jvm
jstarc src/main.sl --target=js
jstarc src/main.sl --target=wasm
```
